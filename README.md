# Erste Segmentierung – Oxford Pets

Dieses Projekt ist ein erstes kleines Testprojekt im Rahmen meiner Bachelorarbeit.
Ziel ist es, eine **Bildsegmentierung** anhand des **Oxford-IIIT Pet Dataset** zu testen und zu verstehen.

## Projektinhalt
- Laden und Anzeigen von Bildern und zugehörigen Segmentierungs-Masken
- Erste Experimente zur Vorbereitung einer späteren ML-basierten Segmentierung
